## Affected Flows

- **Auth flow**: login → JWT cookie + refresh token → /auth/me for identity → logout clears cookies
- **RFQ intake flow**: email/slack → classify → create RfqIntake + Rfq → extract items + customer → match to catalogue
- **Quote workflow**: AI generate → save draft → pending approval → approve → sheet export + email send
- **Connector flow**: DB-registered connectors iterated by cron; env var fallback for legacy
- **Background jobs**: @Cron polls BackgroundJob table every 5s, dispatches to registered handlers
