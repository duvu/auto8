- **`jobs.service.ts`**: `processPendingJobs()` filters by `nextRunAt IS NULL OR <= now`; on failure sets `nextRunAt = now + min(5000 * 2^(attempts-1) + jitter, 300000)ms`; on completion/failure fires `auditService.log(...)` if injected
